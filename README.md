# web-eyes
The 18 lines of code implement a Web fingerprint identification Web Server

### use
``` git clone https://github.com/wujunze/web-eyes.git```     
``` cd web-eys & npm install ```     
```node index.js ```     

```http://127.0.0.1:8080/?url=https://baidu.com```
